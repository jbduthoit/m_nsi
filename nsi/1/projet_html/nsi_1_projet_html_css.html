Projet NSI : Mon premier site web !

Objectifs

Vous devez créer un mini site web en respectant les contraintes ci-dessous:

Attention, il faudra respecter les contraintes imposées.

Vous serez évalué sur les critères suivants :

  • Respect des contraintes imposées ci-dessous.
  • Contenu des pages.
  • Travail sur le code html, css.

Rendre le projet

Vous enverrez par mail votre dossier compressé (dossier contenant l'ensemble des fichiers.) Ce dossier compressé au format .zip sera nommé nom_prenom_projet1.

Il s'agit d'un travail personnel !

Votre dossier doit avoir la structure suivante : /nom_prenom_projet1 --- /images --- --- # toutes les images sont dans le dossier "images" --- index.html --- passion.html --- soures.html --- > éventuellement d'autres fichiers html

Les fichiers et dossier présents

Votre site contiendra au minimum les pages suivantes :

  • la page index.html
  • la page passion.html (celle qui a été réalisée pendant le cours et que vous pouvez encore améliorer)
  • la page sources.html

Le css

Le css de vos pages est à déclarer dans le fichier style.css. On ne devra trouver aucun style css dans les pages html. Les pages html sont liées à cette feuille de style css par la présence dans la partie head du code <link rel="stylesheet" href="style.css">.

Le css actuellement présent n'est là que pour vous donner des exemples. Consultez internet pour obtenir de l'aide !

Évitez de surcharger vos pages par des couleurs ou des images en excès. Il est préférable de rester sobre.

Votre code.

Le code html de chaque page doit être indenté proprement de façon à ce que le début et la fin de chaque élément apparaisse clairement.

Le code html devra être vérifié en chargeant vos pages sur le w3 validator. Il ne devra rester aucune erreur.

Le code css est à contrôler également sur : w3 css validator.

Structure des pages

Chaque page devra contenir :

  • Un header
  • Une partie principale
  • Un footer

Partie header

La partie header contiendra le titre de la page et une image de fond (à vous de modifier celle qui est actuellement présente).
