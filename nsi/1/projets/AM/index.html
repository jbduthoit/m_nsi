La création de Python

Son créateur

Son créateur est Guido van Rossum.
Né a Haarlem (Pays-Bas) en 1956, il étudie au Centre de mathématiques de l'université d'Amsterdam et obtient son Master en 1982.
Guido van Rossum a travaillé à :

  • SARA (1977-1982)
  • ABC Group (1982-1986)
  • Amoeba Project (1986-1991)
  • CWI (1991-1995)
  • CNRI (1995-1998)
  • BeOpen.com (2000)
  • Zope Corporation (2000-2003)
  • Elemental Security (2003-2005)
  • Google (2005-2012)
  • Dropbox (2013-2019)

Il a créé Python pendant qu'il travaillait chez CWI
Il travaille actuellement chez Microsoft depuis Novembre 2020.

Son origine

Le langage Python a été créé car les appels système du système d'exploitation Amoeba, sur lequel il travaillait dans une équipe, sont difficilement interfaçables avec le Bourne Shell qui était la référence de l’époque, utilisé comme interface utilisateur.
En 1989, pendant une semaine de vacances de Noël, il a créé la première version de son langage, inspiré par le langage ABC qu'il connaissait bien car il avait participé à la création de ce langage.
Le nom de ce langage a été choisi en hommage aux Monty Python, une troupe de comiques britanniques.
La première version de Python est apparue le 20 février 1991.
La version 0.9.1 de Python est toujours accessible; elle date de 1991.

L'ascension de Python

Années 1990-2000

Dans les années 1990, Python était un petit acteur, même au sein de la catégorie des langages dynamiques. Le grand acteur, c’était Perl.
A la fin des années 1990, des personnes voulaient utiliser des bases de code écrites en C ou C++, c’est alors qu’est né Numeric Python (NumPy). Un mouvement s'est lancé et des utilisateurs ont ajouté des outils de visualisation de données (matplotlib).
Des chercheurs ont commencé à s'intéresser à Python grâce a IPython et IPython Notebook qui leur permettait de manipuler de grandes quantités de données.
Toute la communauté Python était impliquée dans l'Open source; les développeurs améliorent ce qu'ils aiment et partagent le résultat en Open source. C’est une philosophie commune.
Enfin, dans les années 2000, Python a pris le dessus sur Perl.

Python contre Perl

Python est beaucoup plus facile à apprendre que Perl. Perl remplace avantageusement les scripts Unix, car ce langage utilise beaucoup d'éléments issus de ce système, donc apprendre ce langage requiert une connaissance d'Unix.
Python requiert la connaissance de mathématiques, niveau lycée et quelques clefs du langage, donc la communauté scientifique a pu l'utiliser sans problèmes.

2010 à aujourd'hui

Python est devenu le langage le plus recherché sur Stack Overflow.
Et d'après les tendances Google, Python a pris le dessus sur tous les autres langages depuis 2013.
Python est le langage principal enseigné dans les lycées en France.
Le secteur du Big Data a adopté Python comme l’un de ses principaux outils.

Les caractéristiques de Python

Compatibilité avant tout

Chaque version d'une branche Python (1, 2, 3) peut lire le code écrit pour les anciennes versions de cette branche.
Exemple : du code pour Python 3.7.10 est compatible avec Python 3.9.1 alors que Python 3.9.1 a été créé avant Python 3.7.10 (mais Python 3.9 a été créé après Python 3.7).
Le passage entre deux versions majeures peut être difficile (Python 2 --> Python 3). Certains programmes sont toujours en Python 2 alors que cette branche n'est plus supportée.
Néanmoins, les développeurs peuvent développer des scripts sans trop se soucier de la compatibilité avec les nouvelles versions de Python, ce qui a permis d'attirer les grandes entreprises et développeurs.
Python est un langage interprété donc il est compatible avec tous les systèmes d'exploitation.

Syntaxe et grammaire du langage

La grammaire complète du langage est détaillée ici.
Une particularité de Python est l'indentation. Elle est primordiale en Python, sinon le programme ne fonctionnera pas, ce qui n'est pas le cas de la plupart des langages donc l'objectif du langage Python est d'être lisible.
La création de classes est très facile en Python.
