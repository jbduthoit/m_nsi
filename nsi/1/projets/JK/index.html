Définition du WEB :

Le WEB est le terme utilisé pour parler du world wide web (WWW) ou en Français : toile d'araignée mondiale. Celui-ci est un systeme qui utilise internet pour naviguer entre des documents par des liens hypertextes. Pour se rendre sur une page WEB, il faut utiliser un moteur de recherche(google) et un navigateur WEB(Chrome). Le navigateur WEB permet d'afficher la page WEB et le moteur de recherche est nécessaire pour chercher une page WEB.

Attention, il ne faut pas confondre le WEB et internet. Internet est un réseau permettant de relier les ordinateurs entre eux et donc de les faire communiquer.

Création du WEB :

  1. Le WEB a inititalement été inventer par Tim Berners-lee en 1989 afin que les collaborateurs du CERN (l'Organisation européenne pour la recherche nucléaire) puissent partager des informations entre-eux.L'ingénieur Robert Cailliau a fortement contribué à la diffusion du WEB. Tim Berners-lee a également créée le premier naviguateur WEB qui est aussi le premier éditeur WEB. Ce dernier est intitulé World Wide Web. Il rend ce projet public le 6 aout 1991.
  2. En 1993, le CERN permet l'implantation du World Wide Web dans le domaine public.
  3. Lors de la même année, Eric Bina et MArc Andreessen créent NCSA Mosaic. Ce dernier est un navigateur WEB. Il est le premier à intégrer des images dans les pages WEB. Cela accroit fortement la popularité du WEB.Netscape est ensuite créer par des anciens développeurs de NCSA Mosaic. Aprés de nombreuses évolutions, celui-ci deviendra Firefox.
  4. En 1995, Microsoft lance son propre navigateur WEB: internet explorer afin de concurrencer NCSA. Cela va démarrer la "Guerre des navigateurs". Grâce à cela, les navigateurs vont étre ameliorer trés rapidement.
  5. En 2008, Chrome est lancé. 8ans plus tard,en 2016, il devient le navigateur le plus utilisé au monde en dépassant Internet explorer.
