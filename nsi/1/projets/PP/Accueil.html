Présentation

Bienvenue sur mon mini site web codé en html et en css !

Ce mini site web contient quatres pages :

  • Une page d'accueil sur laquelle vous vous trouvez
  • Un Curriculum Vitae
  • Histoire
  • Une page ou se trouve les sources que j'ai utilisées

La page d'accueil

Cette page ne sert pas à grand chose mais elle me permet de présenter rapidement le site et de pouvoir organiser mon site à partir de cette page.

Le Curriculum Vitae

Mon CV contiendra les informations suivantes :

  • Mon nom, mon prénom et ma photo
  • Les trois spécialités que j'ai choisi
  • Les deux spécialités que je pense garder l'an prochain.
  • Ce que je souhaites faire après le bac
  • Mes centres d'intérêt

Histoire du html

Cette page contiendra les recherches que j'ai pu faire sur l'histoire du langage html.

Les Sources

Cette page répertorie les sources dont j'ai eu besoin nottament pour comprendre certaines balises ou pour savoir comment je pouvais obtenir le rendu que je voulais.
