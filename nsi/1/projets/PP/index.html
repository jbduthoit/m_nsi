Histoire du language HTML


Le language html est un language de balisage basé sur le SGML(Standard Generalized Markup Language) notamment utilisé pour la création de page web

Les débuts du html

Le html fut créé en 1992 par Tim Berners-Lee,le principal inventeur du World Wide Web. C'est l'une des 3 inventions (avec le HTTP et les URL) qui à grandement contribué à la création du World Wide Web
Les éléments que comprend l'html lors de sa création sont :

  • Le Titre du document
  • Les liens hypertext
  • La structuration du texte (titre, sous-titre...)
  • Une recherche par index

Suite à la création du html il va y avoir un effort de normalisation, cepandant jusqu'à la fin des années 90 le html va être surtout défini par les spécificité qu'apporte le navigateur.
Par exemple le navigateur NCSA mosaic à permit d'intégrer des images et des formulaires(espaces de saisie) ce qui a rendu le Web interactif. Le Naviagteur Netscape Navigator ajouta des éléments de présentation tel que les attributs de texte,le centrage...


Les concepteurs décident d'utiliser un language de style pour la présentation : Les Feuilles de style en cascade(CSS) qui fut implémenté rapidement dans les navigateurs pour améliorer significativement la présentation des page


>

Html 3.2 et 4.0


En janvier 1997 la W3C(World Wide Web Consoortium) présente le html 3.2 qui avec les ajouts d'internets explorer et de Netscape Navigator propose une normalisation des tables et des nombreux éléments e présentation
Peu après en décembre 1997 est publié la spécification 4.0 qui en plus d'apporter des amélioration sur l'accessibilité du contenu apporte une standardisation de nombreuses extensions supportant les style, les script, les cadres et les objets


Le html 4.0 comprend 3 variantes :

  • la variante stricte qui exclu des éléments de présentation qui seront remplacé pat les styles CSS
  • la variante transitoire qui étend la variante stricte au élément et attributs peu apprécier du html 3.2
  • la variante frameset qui normalise le jeu de cadre composant une ressources unique à aprtir de plusieurs page web

L'XHTML


Le devellopement du html est abandoné au début des années 2000 au profit du XHTML. Mais en 2004 des éditeurs de navigateurs web (WHATWG)veulent relancer le developpement du html qui répondrai mieux aux nouveaux besoins que l'XHTML 2.0 qui est en cours de conception. Par la suite le XHTML sera abandoné au profit d'une nouvelle spécification du html : l'html 5.


L'html 5 de 2007 à aujourd'hui

Suite aux plaintes d'une partie de l'industrie du web la W3C relance le devellopement du html en mars 2007. En Mai 2007 les travaux du WHATWG sont accepté comme base pour la nouvelle spécification du html comportant les principes suivants :

  • la compatibilité du html avec le contenu web déjà existant : pouvoir exploiter des futur contenu html 5 sur des agents plus ancien
  • préferer les évoliution plutôt que des changements radicaux et l'adoption des technologies pratique approuvé par les auteurs de contenus actuels
  • donner plus d'importance aux besoins de l'utilisateurs qu'aux besoins des auteurs et/ou au besoin des navigateur

Finalement la W3C abandonnes le développement du XHTML 2.0 en 2009 car celui-ci est devenu obsolète avec l'implantation du XML(extensive Markup language) dans le html 5.
